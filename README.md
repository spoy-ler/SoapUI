# SoapUI
Testing the SOAP API.
<br>
Using the web service description at [DataFlex Web Service for Country information](http://webservices.oorsprong.org/websamples.countryinfo/CountryInfoService.wso) (WSDL can be downloaded at [CountryInformation.wsdl](http://webservices.oorsprong.org/websamples.countryinfo/CountryInfoService.wso?WSDL)), was created a test set:

4 test steps with different data for positive and negative testing with at least 3 asserts and 2 variables:

- CapitalCity
- CountryISOCode
- CountryCurrency
- CountriesUsingCurrency
